# Prism-CRISPR
Prism-CRISPR: A Deep Learning Architecture for sgRNA Prediction. This framework includes several core models: Prism-on is used to predict the on-target cleavage efficiency of Cas9 sgRNAs, while Prism-be is designed to predict base editing outcome distributions for base editors (ABEs/CBEs).
